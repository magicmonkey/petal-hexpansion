A "petal" hexpansion for the EMFcamp Tildagon badge.

It has 3 bands of LEDs which are individually controllable via I2C, using the AL5887 chip.

PCB schematics are in this repo, and the badge app software is in https://github.com/magicmonkey/petal-hexpansion-software

Use the "Petals" app in the badge app store to see some LED animations.
